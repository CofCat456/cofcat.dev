# cofcat.dev

![Blog homepage](./docs/homepage.png)

CofCat personal website

Visit: [https://cofcat-dev.vercel.app/](https://cofcat-dev.vercel.app/)

## Features

- Next.js 13
- ContentLayer (loading local MDX files)
- TailwindCSS
- Headlessui
- Eslint, Prettier

## Commands

### Start local dev server

```bash
yarn dev
```
